# Dynamic Rate Limits

See how to create dynamic rate limits with the Rick & Morty API sample.

## Key Files

| File | Purpose |
|------|--------|
| `config/routes.oas.json` | Route definitions with policies |

## Getting Started

**Create a local copy:**
```bash
npx create-zuplo-api@latest --example dynamic-rate-limits
```

**Run locally:**
```bash
npm install
npm run dev
# Server runs on https://localhost:9000
```

**Deploy to Zuplo:**
```bash
zuplo deploy
```
Or use the [Deploy to Zuplo](https://zuplo.com/docs/examples/dynamic-rate-limits) button.

## Environment Variables

None required for this example.
